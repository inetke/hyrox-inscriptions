# HYROX Event Registration Management Platform

## Overview

This project was developed after identifying operational challenges in the registration process for HYROX or similar events organized by the training center where I train.

As event participation grew, organizers were managing registrations, participant communication, and event capacity manually, primarily through WhatsApp and spreadsheets. This created a significant administrative burden and increased the risk of errors.

To address these challenges, I designed and developed a web application that centralizes event registrations, automates administrative workflows, and provides organizers with real-time visibility into participant management.

The application was initially developed as a proof of concept and evolved into a practical solution capable of supporting real event operations.

## Business Problem

The organizers were experiencing several operational issues:

- High volume of repetitive WhatsApp inquiries.
- Manual participant registration management.
- Difficulty tracking event capacity.
- Limited visibility of payment status.
- Time-consuming participant communication.
- Risk of registration errors and duplicated records.

The goal was to reduce administrative workload while improving the registration experience for participants.

---

## Solution

The platform centralizes the entire registration process through a web-based application where participants can:

- Register for available HYROX events.
- Select event categories.
- Submit personal information.
- Receive automated confirmations.
- Access event information in a structured format.

For organizers, the application provides:

- Real-time participant management.
- Event capacity control.
- Registration monitoring.
- Participant database management.
- Automated communication workflows.
- Administrative dashboards and reporting.
